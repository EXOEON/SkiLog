SkiLog: Bear Valley is a tool to keep track of all the runs you have done at Bear Valley and hopefully know when you have done them all. It shows percentages and saves the data through an account system. You data is fully protected and you do not need to provide an email adress or any other personal information.

Features

  • Account System
  • Frontside, Backside, village, and lower mountain runs
  • Run difficulty
  • Run completion
  • Percentage completion
  • Responsive design

This is an unofficial tool and is for personal use only.
